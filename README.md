# 🛰️ sat-track

> Веб-приложение для отслеживания спутников на орбите Земли в реальном времени.

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![HTML5](https://img.shields.io/badge/HTML-100%25-E34F26?logo=html5)]()

## 📖 Описание

**sat-track** — это легковесное веб-приложение, которое позволяет визуализировать положение искусственных спутников (МКС, Starlink и др.) на карте Земли. Проект использует TLE-данные (Two-Line Elements) для расчёта орбит и отображения позиций в браузере.

## ✨ Возможности

- 🌍 Отображение спутников на интерактивной карте
- 🛰️ Поддержка отслеживания популярных спутников (МКС, Starlink, Hubble)
- 🔄 Обновление данных в реальном времени
- 📍 Прогноз пролёта над заданной локацией
- 📱 Адаптивный дизайн для мобильных устройств
